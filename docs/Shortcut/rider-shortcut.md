# Rider 快捷鍵

## In Windows

會使用快捷鍵

|快捷鍵         |功能           |備註           |
|:-------------:|:-------------:|:------------:|
|Alt+F5     |Debug Run       |               |
|Shift+F5     |Stop Run       |               |
|Ctrl+Alt+L     |Explorer       |               |
|Ctrl+Alt+S     |Settings       |               |
|Ctrl+Alt+1     |Open Terminal       |               |
|Ctrl+Alt+3     |資料庫Tools       |               |
|Ctrl+Shift+F9     |刪除所有中斷點       |               |
|Ctrl+Shift+O     |開啟專案       |               |
|Ctrl+Shift+F     |關鍵字查詢       |               |
|Ctrl+Shift+T     |Go to File       |               |
|Ctrl+D          | 重複目前所在行/選取文字 | Edit.Duplicate  |
|Ctrl+G     |跳到指定行數       |               |
|Ctrl+Tab     |切換正在編輯程式       |               |
